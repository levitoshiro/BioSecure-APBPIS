# 🛡️ BioSecure Framework (Full-Stack Deployment)

**BioSecure** is a zero-trust Electronic Health Record (EHR) management framework. This repository contains the complete, production-ready architecture, integrating a local Ethereum blockchain (Ganache/Hardhat), a Python Flask Global Node (Backend), and a React Web3 interface (Frontend).

## 🏗️ System Architecture

To run this application locally, you will need to run three separate environments simultaneously. 

**Prerequisites:** Ensure you have [Node.js](https://nodejs.org/) and [Python 3](https://www.python.org/) installed on your machine.

---

## 🚀 Step 1: Blockchain Network (Terminal 1 & 2)
*This step initializes your local Ethereum blockchain and deploys the BioSecure Smart Contracts.*

Open a new terminal, navigate to your blockchain folder, and install the dependencies:
```bash
cd <your-blockchain-folder>
npm install
npm install -g ganache

Start the Local Blockchain (Terminal 1):
Leave this terminal running to keep the blockchain active.
Bash

ganache -p 7545

Deploy the Smart Contract (Terminal 2):
Open a second terminal in the same blockchain folder. Compile and deploy your contract to the Ganache network.
Bash

npx hardhat compile
npx hardhat run scripts/deploy.js --network ganache

    ⚠️ Important: After deployment, the terminal will output a Contract Address. Copy this address, as you will need to paste it into your Frontend's .env file!

🐍 Step 2: Global Node / Backend (Terminal 3)

This step starts the Python Flask server that acts as the decentralized ledger's state manager.

Open a third terminal and navigate to your backend folder:
Bash

cd <your-backend-folder>

(If you have a virtual environment or requirements file, activate/install it here). Then start the server:
Bash

python app.py

Leave this terminal running.
💻 Step 3: Web3 Frontend (Terminal 4)

This step installs and starts the React user interface.

Open a fourth terminal and navigate to your frontend folder. If you haven't created it yet, run the Vite creation command, otherwise skip to npm install:
Bash

# Only if you haven't created the frontend folder yet:
# npm create vite@latest frontend -- --template react

cd frontend
npm install
npm install ethers axios

Start the development server:
Bash

npm run dev

🦊 Step 4: MetaMask Configuration

To interact with the application, you must configure your MetaMask browser extension to connect to your local Ganache blockchain.

    Open MetaMask and enter your password.

    Go to Settings > Networks > Add Network > Add a network manually.

    Enter the following exact details:

        Network name: Localhost 8545

        New RPC URL: http://127.0.0.1:7545

        Chain ID: 1337

        Currency symbol: ETH

    Click Save.

    Optional but recommended: Import one of the private keys generated by your Ganache terminal into MetaMask so you have test ETH to pay for gas fees.
